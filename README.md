## 使用

1.專案說明:個人記帳本，使用者須先創建自己帳號才可以使用。

2.使用者可新增、編輯、刪除每筆帳目，並依照類別篩選帳目

## 安裝/啟動
開啟終端機
git clone到本地端
```
$git clone https://github.com/zxc58/expense-tracker.git
```
進入專案資料夾
```
$cd expense-tracker
```
安裝必要套件
```
$npm install
```
啟動
```
$npm start
```
